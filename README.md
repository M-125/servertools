# Tools made for my home server
## minecraft.py
joining while the "minecraft-server" service isn't running starts it and sends a kick message
## leds.py
Manipulate leds from /sys/class/leds
Can be used for displaying running services on keyboards thru the capslock, numlock, or scrolllock lights
## services.py
Manage systemctl services
start/stop them
or get their status
## test.py
led tester

# Leds.py and services.py doesn't work without root privileges or their specific permission groups
